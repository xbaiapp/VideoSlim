## 6. 停止条件

出现任一情况立即停止并提交素材信息、步骤、截图/录屏和完整 F19 日志：

1. 0°/90°/180°/270° 任一样本的框选与输出出现无法由 ≤2px 取整解释的错位；
2. 画面方向错误、双重旋转、拉伸或 `Crop → Presentation` 顺序不符；
3. App 卡死、任务永不结束、重复任务、错误成功、错误取消或无法恢复；
4. 源文件或旧输出丢失、被覆盖或修改；
5. VideoSlim 之外的文件被删除；
6. 失败/取消产生用户可见且无法清理的错误文件。

不得改用中间文件后二次有损转码来绕过旋转或裁剪问题。
